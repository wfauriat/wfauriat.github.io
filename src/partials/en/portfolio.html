Full-stack Demonstrator: Bayesian Inference Learning

▾

- Setup and mastery of Full-stack development: Flask/Python backend, React frontend

- Applied to teaching Bayesian inference — hands-on manipulation and visualization

- Two platforms developed: Web (Flask - React), Desktop (PyQt)

Feature details:

  • Choice among several regression cases (synthetic data). Option to import a custom dataset (CSV)
  • Configurable priors per parameter — Uniform, Normal, Half-Normal
  • Choice and tuning of inference algorithm (pure NumPy): Metropolis-Hastings MCMC, MHwGibbs
  • Bayesian regression vs. scikit-learn model comparison (Linear, ElasticNet, RandomForest, SVR)
