Bayesian Inference App

▾

Full-stack educational app for interactive Bayesian inference via MCMC — posterior visualization and classical ML comparison. Deployed in production.

Modular architecture with a NumPy-only inference engine, a Flask REST API (6 blueprints covering configuration, data, computation, and results), a React frontend with Recharts visualizations, and a parallel PyQt5 desktop version.

  • Metropolis-Hastings MCMC with adaptive proposal tuning during burn-in
  • Configurable priors — Uniform, Normal, Half-Normal per parameter
  • Direct Bayesian vs. scikit-learn model comparison (Linear, ElasticNet, RandomForest, SVR)
  • Custom CSV data upload alongside pre-built cases (polynomial regression, California housing)
  • Dual interface — web app accessible anywhere, desktop app for offline use

UQ Initiation App

▾

Desktop tool for learning uncertainty quantification — interactive simulations with a standalone PyQt GUI.

Standalone PyQt5 application designed as an accessible entry point into UQ concepts. Bridges the gap between statistical theory and practical application through guided, interactive workflows.

  • Interactive interface for practitioners and learners alike
  • Covers core UQ methods — propagation of uncertainty through models
  • Self-contained: no external server or backend required
