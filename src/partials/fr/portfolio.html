Demonstrateur Full-stack : Apprentissage de l'inférence Bayésienne

▾

- Mise en place et maîtrise d'un développement Full-stack : Backend en Flask/Python, Frontend en React

- Application à l'enseignement de l'inférence Bayésienne - manipulation et visualisation

- Développement de deux plateformes : Web (Flask - React), Desktop (PyQt)

Détails sur les fonctionnalités :

  • Choix parmi plusieurs cas de regression (données synthétiques). Possibilité d'importer un jeu de données (CSV)
  • A priori configurables par paramètre — Uniforme, Normal, Demi-Normal
  • Choix et réglage de l'algorithme d'inférence (pur Numpy) : Metropolis Hastings MCMC, MHwGibbs
  • Comparaison regression Bayésienne vs. modèles scikit-learn (Linear, ElasticNet, RandomForest, SVR)
