Application d'Inférence Bayésienne

▾

Application full-stack éducative pour l'inférence bayésienne interactive via MCMC — visualisation de la distribution a posteriori et comparaison avec des méthodes ML classiques. Déployée en production.

Architecture modulaire avec un moteur d'inférence basé uniquement sur NumPy, une API REST Flask (6 blueprints couvrant configuration, données, calcul et résultats), un frontend React avec visualisations Recharts, et une version desktop parallèle en PyQt5.

  • MCMC Metropolis-Hastings avec ajustement adaptatif de la proposition pendant le burn-in
  • A priori configurables — Uniforme, Normal, Demi-Normal par paramètre
  • Comparaison directe Bayésien vs. modèles scikit-learn (Linear, ElasticNet, RandomForest, SVR)
  • Upload de données CSV personnalisées en plus de cas préconstruits (régression polynomiale, logement Californie)
  • Interface double — application web accessible partout, application desktop pour usage hors ligne

Application d'Initiation à la Quantification d'Incertitude

▾

Outil desktop pour l'apprentissage de la quantification d'incertitude — simulations interactives avec une interface PyQt autonome.

Application PyQt5 autonome conçue comme un point d'entrée accessible aux concepts de quantification d'incertitude. Comble le fossé entre la théorie statistique et l'application pratique à travers des workflows guidés et interactifs.

  • Interface interactive pour praticiens et apprenants
  • Couvre les méthodes UQ essentielles — propagation d'incertitude à travers des modèles
  • Autonome : aucun serveur externe ou backend requis
